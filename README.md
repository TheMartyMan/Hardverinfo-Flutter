# Hardverinfo
Eszköz hardveres adatainak megjelenítése Flutter applikáció segítségével

**Valós idejű, részletes információt szolgáltat az alábbiakról:**
- Eszköz
- Akkumulátor
- CPU
- Memória
- Hálózat
- Szenzorok
