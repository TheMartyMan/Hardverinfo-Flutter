# Hardverinfo
Android alapú eszközök hardveres adatainak megjelenítése Flutter applikáció segítségével.

**Valós idejű (konfigurálható), részletes információt szolgáltat az alábbiakról:**
- Eszköz
- Akkumulátor
- CPU
- Rendszer
- Memória
- Szenzorok
- Hálózat
